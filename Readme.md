# Block Game 3D

## Simple game with moving blocks built using three.js

## Stack used
- JS;
- CSS;
- HTML;
- Three.js;
- gsap.

## Running application
Clone repo
```
git clone https://github.com/lu7623/ProgressBlock.git
```
Install dependencies
```
npm install
```
To run in a dev mode
```
npm run dev
```
To build 
```
npm run build
```
